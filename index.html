  1. Estática
  2. Uso de framesets
  3. Guestbooks
  4. Usuarios:
    1. Unicamente como lectores
Web 2.0
  • Abierta y Dinámica
  • Fortalecimiento del trabajo colectivo
  • Amplias aplicaciones para los usuario
  • Usuarios:
    • Producir contenido y opinar
    • Pueden crear espacios propios
Web 3.0
Inteligencia
* Permite clasificar las paginas de internet, un sistema de etiquetado para no solo encontrar información sino que también entenderla.
Sociabilidad
* Las comunidades sociales se hacen más exclusivas y complejas.
Rapidez
* La transmisión de video en la red y el nacimiento de portales dedicados a esa tarea.
Distribución
* Los programas y la información se convierten en pequeñas piezas distribuidas por la Web.
Web 4.0
  1. Comprensión del lenguaje natural(NLU)
  2. Técnicas de Speech-to-Text
  3. Nuevos modelos de comunicación máquina-máquina(M2M)
  4. Uso de información de contexto del usuario
  5. Nuevo modelo de interacción con el usuario
